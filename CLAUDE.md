## Project Overview

This is a NixOS system configuration using Flakes with Home Manager integration. The configuration manages both system-level settings and user-level home directory configuration with a modular architecture supporting cross-platform shared modules.

## Key Commands

### Apply Configuration
```bash
# Apply the full system configuration (requires sudo)
sudo nixos-rebuild switch --flake .#nixos

# Or use the convenience script (uses Tsinghua mirror for faster builds)
./apply.sh
```

### Common Nix Commands
```bash
# Check flake for errors
nix flake check

# Update flake inputs
nix flake update

# View flake outputs
nix flake show

# Build without switching (dry-run)
nixos-rebuild build --flake .#nixos
```

## Architecture

```
nixos-config/
├── flake.nix              # Flake entry point
├── flake.lock             # Locked versions (auto-generated)
├── home.nix               # User global settings (username, version)
├── hosts/                 # Machine-specific configurations
│   ├── nixos/             # NixOS hosts
│   │   └── desktop.nix    # Desktop host config (boot, hostname, timezone)
│   └── darwin/            # macOS hosts (placeholder for future)
├── modules/               # Modular configuration
│   ├── nixos/             # NixOS-specific system modules
│   │   ├── default.nix    # Imports all nixos modules
│   │   ├── docker.nix     # Docker configuration
│   │   ├── nix.nix        # Nix settings, substituters, experimental features
│   │   ├── networking.nix # Network, DNS, firewall, SSH
│   │   └── users.nix      # User accounts, system packages, zsh
│   ├── darwin/            # macOS-specific modules (placeholder)
│   │   └── default.nix
│   └── shared/            # Cross-platform Home Manager modules
│       ├── default.nix    # Imports all shared modules
│       ├── core.nix       # Common packages (htop, tree, rg, fd, nvim, tmux)
│       ├── git.nix        # Git user configuration
│       ├── shell/         # Shell configurations
│       │   ├── zsh.nix    # Zsh + starship + plugins
│       │   └── tmux.nix   # Tmux + plugins
│       └── editors/       # Editor configurations
│           └── nvim.nix   # Neovim (placeholder for config)
├── hardware-configuration.nix  # Auto-generated by nixos-generate-config
├── conf.d/                # Non-Nix config files (optional)
└── apply.sh               # Convenience script with mirror
```

### Module Organization

- **`hosts/`** - Per-machine configurations (bootloader, hostname, timezone, hardware imports)
- **`modules/nixos/`** - NixOS system-level configuration (requires root, system services)
- **`modules/darwin/`** - macOS-specific configuration (for future cross-platform use)
- **`modules/shared/`** - Home Manager user-level configuration (works on both NixOS and macOS)

### Home Manager Integration

Home Manager is integrated as a NixOS module with:
- `home-manager.useGlobalPkgs = true` - Uses system nixpkgs
- `home-manager.useUserPackages = true` - Installs packages to user profile
- User "one" is defined in `modules/nixos/users.nix` (system) and `home.nix` (home config)

### Adding New Configuration

1. **System-level (NixOS only)**: Add modules to `modules/nixos/` and import in `default.nix`
2. **User-level (cross-platform)**: Add modules to `modules/shared/` and import in `shared/default.nix`
3. **New machine**: Create `hosts/<hostname>.nix` and add to `flake.nix`

### Important Details

- Uses nixos-25.11 channel
- Substituters configured for Chinese mirrors (USTC, Tsinghua)
- User shell is zsh with starship prompt, autosuggestions, and syntax highlighting
- Docker is enabled and starts on boot
- systemd-boot bootloader (UEFI)
